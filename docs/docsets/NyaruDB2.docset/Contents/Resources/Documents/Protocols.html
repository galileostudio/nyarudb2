FieldValueConvertible

  • FieldValueConvertible

    A protocol for types that can be converted to a FieldValue for use in queries and index operations.

    Conforming types include String, Int, Int64, Double, Bool, UUID, Date, and Optional wrappers. This allows the public API to accept a wide range of value types without forcing callers to construct FieldValue directly.
